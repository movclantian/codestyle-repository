# Logo 生成提示词模板

## 模板说明

这是一个用于生成高质量产品Logo设计提示词的模板，基于FreeMarker（.ftl）格式，支持变量替换，可以快速生成不同产品的Logo设计需求。

## 模板文件

- `logo-prompt.ftl`: Logo生成提示词模板，包含可替换变量

## 变量说明

| 变量名 | 类型 | 描述 | 示例 |
|-------|------|------|------|
| `productName` | String | 产品名称 | Chrome助手 |
| `productType` | String | 产品类型/功能描述 | 简单自动补写助手 |
| `style` | String | 设计风格 | 扁平、简约现代 |
| `keyConcept` | String | 核心概念 | 智能助手、自动补写 |
| `mainColor` | String | 主色调 | 蓝色 |
| `secondaryColor` | String | 辅助色调 | 白色/灰色 |

## 使用示例

### 原始模板
```
请为我生成${productName}(${productType})的Logo设计：

设计要求：
- 尺寸比例：1:1，推荐尺寸：48X48、64X64、128X128
- 设计风格：${style}，现代化、简约、专业
...
```

### 替换后（以Chrome助手为例）
```
请为我生成Chrome助手(简单自动补写助手)的Logo设计：

设计要求：
- 尺寸比例：1:1，推荐尺寸：48X48、64X64、128X128
- 设计风格：扁平、简约现代，现代化、简约、专业
...
```

## 特点

- **通用性**：适用于各类产品Logo设计
- **可定制性**：通过变量轻松调整设计要求
- **标准化**：统一的设计规范确保输出质量
- **易用性**：简单明了的结构便于理解和使用

## 注意事项

1. 建议根据实际需求调整变量值，特别是产品特性和设计风格
2. 对于特定行业或产品，可以扩展模板添加更多行业相关的设计要求
3. 生成的提示词可以直接用于AI设计工具（如Midjourney、DALL-E等）
